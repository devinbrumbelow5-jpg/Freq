# Aether Trading Bot - 2-Hour Stress Test & Critical Bug Fixes

## Session Date: March 26, 2026

### Critical Bugs Fixed

1. **v5.8 Signal Handler Bug**
   - Issue: Bot was exiting on transient signals (SIGHUP, etc.)
   - Fix: Modified shutdown() handler to only exit on SIGTERM/SIGINT
   - File: `/root/AetherTrader-v5/aether_v5_8.py`

2. **systemd Watchdog Killing Bot**
   - Issue: `WatchdogSec=60` was killing the bot every 60 seconds because bot doesn't implement watchdog protocol
   - Fix: Disabled `WatchdogSec` in systemd service file
   - File: `/etc/systemd/system/aether-bot-pure.service`

3. **Database Schema Mismatch**
   - Issue: Missing columns in SQLite schema (entry_momentum, entry_rsi, bot_name)
   - Fix: Added all missing columns via ALTER TABLE
   - File: `/root/AetherTrader-v5/data/aether.db`
